Quiénes somos?

ISQM Information System Quality Monitoring

Es un sistema de información el cual mejorara el proceso de monitoreo de chat de servicio de Open English este se apoyará en las muestras obtenidas de las conversaciones entre el asesor de chat y el estudiante. Con el consolidado de estas muestras el SI escalara esta información al Gerente de operaciones para que este parametrize nuevos atributos de calidad y retroalimente procesos, para que mejore la calidad de la información que se brinda a los estudiantes.
